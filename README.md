# Sistema de Punto de Venta

- **Repositorio:** https://github.com/Shvy-bit/ProyectPersonal
- **Proyecto hospedado:** https://shvy-bit.github.io/ProyectPersonal/

Este proyecto implementa un sistema de ventas para una distribuidora de alimentos balanceados, con módulos de ventas, cambio de cajero, administración de cajeros, inventarios y facturación, accesibles desde una interfaz web.

**Tecnologías utilizadas:** HTML, CSS, JavaScript, Python (Flask) y SQL (MySQL).
